## 系统要求
- Linux 或 macOS 系统
- 至少 64GB RAM
- 至少 50GB 可用磁盘空间
- 稳定的网络连接

## 快速开始

### 1. 下载安装脚本
```bash
wget https://raw.githubusercontent.com/zorp-corp/nockchain/main/install.sh
chmod +x install.sh
```

### 2. 运行安装脚本
```bash
sudo ./install.sh
```
